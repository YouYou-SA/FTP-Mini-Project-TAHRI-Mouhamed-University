# FTP-Mini-Project-TAHRI-Mouhamed-University
A Streamlit-powered web dashboard that recommends optimal crops based on soil type, pH, EC, moisture, and temperature... . Enter your soil conditions and get instant crop recommendations with probability scores. Perfect for farmers, agricultural consultants, and agri-tech prototyping. Simple and fast.
Crop Recommendation ML Dashboard

Interactive Streamlit app for crop recommendation using machine learning. Predict optimal crops based on soil, temperature, humidity, and other factors.

🚀 Quick Start (2 minutes)
Prerequisites
Python 3.8+
. Run Locally
streamlit run app.py
App opens at: http://localhost:8501

📊 Features
Interactive crop prediction
Soil type analysis
ML model visualization
Responsive dashboard
